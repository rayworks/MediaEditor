# MediaEditor

The project shows the usage of merging mutiple audio files
via [FFmpegAndroid](https://github.com/WritingMinds/ffmpeg-android-java).

## Setup 
Use `adb push` to copy all the testing files from `res/raw` folder to `/sdcard/`
on your device.
